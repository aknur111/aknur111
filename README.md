## Featured projects

### AI Tender Platform
> `LangChain` `OpenAI API` `FastAPI` `PostgreSQL` `Celery` `Redis` `WebSocket` `Telegram Bot`

AI system that automatically parses Kazakhstan government procurement documents (`goszakup.gov.kz`), extracts key requirements via LLM, and auto-generates tender proposals based on company profile matching. Real-time WebSocket notifications + Telegram bot for subscribers.

**Key results:** semantic search via FAISS embeddings · async task queue handling 1000+ docs/day · auto-scoring of tender fit

---

### Clinic Appointment System
> `Golang` `Gin` `PostgreSQL` `Redis` `Docker` `JWT`

Full-featured clinic booking platform with role-based access (patient / doctor / admin), real-time slot availability via Redis caching, and email/SMS notification service.

**Key results:** clean architecture with DI · containerized via Docker Compose · middleware for auth, logging, rate limiting

---

### Vehicle Cleanliness Detection
> `YOLOv8` `Python` `OpenCV` `FastAPI` `MLflow`

Fine-tuned YOLOv8 to classify vehicles as clean/dirty. Built a batch inference API for fleet analysis and full data augmentation + model evaluation pipeline.

**Key results:** 92%+ mAP · inference API with batch processing · experiment tracking in MLflow

---

### E-commerce AI Assistant
> `LangGraph` `OpenAI API` `FastAPI` `Redis` `WebSockets`

Multi-tool LLM agent for product search and order tracking. Persistent conversation memory via Redis, real-time response streaming, and fallback strategies for reliable multi-step workflows.

---

### Stock Signal Prediction Model
> `Scikit-learn` `Pandas` `NumPy` `Matplotlib`

Ensemble ML model for buy/hold signal prediction with technical indicator feature engineering and walk-forward backtesting validation.
